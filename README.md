# 🧾 Jednoduchá PHP aplikace – Správce uživatelů

Tato jednoduchá webová aplikace ukazuje základní prvky PHP:

- Proměnné
- Operátory
- Podmínky (if/else)
- Smyčky (foreach)
- Pole (array)
- Funkce
- Práce s MySQL databází přes phpMyAdmin
